# Ingenieria y Calidad de Software (ISW)

Este repositorio se usa para estudiar ISW y construir resumenes a partir de:
- PPTs de catedra
- transcripciones de clase
- bibliografia recomendada al final de cada PPT
- resumenes y apuntes complementarios

## Objetivo
Construir resumenes claros, completos y utiles para parciales escritos.

## Como se trabaja en este repo
- El PPT define el tema, el titulo y el marco general.
- El resumen final no se limita al PPT.
- Cada tema se integra con: PPT + clase + bibliografia recomendada + fuentes complementarias.
- No se asume una jerarquia rigida entre fuentes.

## Parciales
- La materia tiene 2 parciales escritos.
- Los temas de cada parcial no coinciden necesariamente con una unidad completa.
- El foco actual del repositorio es el parcial 1.

## Estado operativo del repo (2026-04-20)
- `fuentes/` ordenado por parcial y por PPT.
- `bibliografia/` separada por parcial/tema, con mapeo por PPT y por fuente.
- `resumen/` mantenido por parcial (no por unidad).
- `inbox/` queda como bandeja de entrada para nuevo material.
